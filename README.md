# Simple To Do List

This is a console-based program written in Java.

The program allows users to manage a simple to-do list through a menu-driven interface. Users can add tasks, view all tasks, and delete tasks from the list.

## Features

- Add tasks to a list
- View all current tasks
- Delete tasks by selecting their number
- Menu-based navigation system
- User input handling using Scanner
- Basic error handling for invalid task selection

## How to Run

Compile:
javac Main.java

Run:
java Main
